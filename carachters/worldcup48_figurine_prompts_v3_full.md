## What changed and how to use it
v3 keeps the v2 individual-person and kit system, with the requested fixes baked directly into each prompt: solid black background, 2K output, clean smiles, slim lean male builds, no bodybuilder physiques, happier positive expressions, and a diversified no-knee-slide pose bank. Kits stay unchanged per nation.

**Reference setup (important):**
- Attach ONLY your original lineup image (the row of figurines) as the **style** reference — for texture, lighting, proportions, finish.
- Do NOT attach a single-character anchor or single hero character as a face anchor. That was copying the face.
- Each prompt says "completely different individual, do not reuse the reference face." Let that do the work.
- Run in Nano Banana Pro. Keep settings identical: full body, tall portrait, solid black background, 2K download. 4 generations per prompt, keep the best.

These figurines represent **fans**, not players — so every character wears the same beanie + flag-patch setup for visual unity, regardless of nation. No religious or state-mandated headwear is imposed on any character.
